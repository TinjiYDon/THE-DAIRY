# THE-DAIRY

离线个人健康日记：隐私优先、多模态记录、温暖陪伴与轻引导。Flutter · Android/iOS · 100% 本地运行。

- 完全离线：不采集、不联网，数据加密存储
- 温暖陪伴：多“人设”语气包（轻松同伴/温柔教练/损友/闺蜜/兄弟/恋人）
- 轻量 AI：本地小模型情感分析 + 规则引擎（MVP 不上 LLM）
- 成长引导：微目标/周回顾/呼吸练习，温柔、不打扰
- 中英双语：中文优先，英文更克制语气

> 仓库与项目主页：[GitHub/THE-DAIRY](https://github.com/TinjiYDon/THE-DAIRY.git)

## 快速开始
```bash
flutter create the_dairy && cd the_dairy
flutter pub get
flutter run
```

## 主要特性
- 日志：文本/图片/打分/标签/提醒、日历与搜索
- 洞察：日/周/月情绪曲线、症状热力图、触发因素 Top N
- 陪伴：记录后暖心卡、低落时“陪伴模式”、微目标与周回顾
- 安全：SQLCipher 加密、加密备份/恢复（手动导出/导入）
- 付费：买断 + 高级包（模板/导出/图表扩展）
